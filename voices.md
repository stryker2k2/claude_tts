# Piper TTS — Available English Voices

To download any voice, run:

```powershell
.\download-piper.ps1 -Voice <voice-name>
```

Quality levels: **low** = 16 kHz / fast, **medium** = 22 kHz / balanced, **high** = 22 kHz / best quality
